# ZezinOS - Portfolio Pessoal

Bem-vindo ao **ZezinOS**!
Este é meu portfólio interativo construído com **HTML, CSS e JavaScript**, simulando a experiência retro de um sistema operacional dos anos 90.

**Acesse o projeto online:** [ZezinOS](https://zezinos.site/)

## Tecnologias Usadas
- **HTML5** (Semântico e estruturado)
- **CSS3** (Estilização)
- **JavaScript (ES6+)** (Modularização com ES Modules, Classes e Eventos)
- **Design System:** Baseado na UI clássica do Windows 98.

## Rodando Localmente

**Obs:** Como este projeto utiliza **ES Modules** (`import` / `export`), ele precisa ser servido via HTTP

### VS Code (Recomendado)
1. Instale a extensão **Live Server**.
2. Clique com o botão direito no `index.html`.
3. Selecione **"Open with Live Server"**.
